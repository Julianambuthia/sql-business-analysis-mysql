# SQL Business Sales Analysis (MySQL)

## Objective

Analyze business sales data using SQL to uncover insights on revenue, customer behavior, product performance, and regional trends.

---

## Tools & Technologies

* MySQL
* MySQL Workbench
* SQL (Aggregation, Group By, Window Functions)

---

## Project Structure

* `schema.sql` — Database and table creation
* `sample_data.sql` — Sample business dataset
* `analysis_queries.sql` — SQL queries for analysis
* `insights.md` — Key business findings

---

## Key Business Questions

* What is total revenue and profit?
* Who are the top customers?
* Which products are most profitable or loss-making?
* How does performance vary by region?
* What trends exist over time?

---

## Key Insights (Summary)

* Technology products generate the highest revenue
* Some furniture products operate at a loss
* A small group of customers drives a large share of revenue
* Sales show clear time-based trends

---

## How to Run

1. Execute `schema.sql` in MySQL Workbench
2. Run `sample_data.sql` to load data
3. Execute `analysis_queries.sql` to perform analysis
